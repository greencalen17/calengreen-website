Machine Learning Solar System Generator

As part of my first project, I aimed to develop a machine learning model that generates solar systems using exoplanet data from NASA Exoplanet Science Institute. Using sklearn's LinearRegression() function, I first implemented a baseline linear regression model to analyze attribute relationships, followed by a more advanced neural network model to enhance solar system generation. The ultimate goal of this model is to assist astronomers in simulating different solar system possibilities and refine incomplete system definitions. This project showed me how even simple ML models can uncover meaningful structure in complex, high-dimensional data. It also extended my understanding and interest in systems that blend scientific data with generative Artificial Intelligence approaches. View presentation here!
